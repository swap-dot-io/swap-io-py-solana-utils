# Solana utils

Utilities for working with Solana

## Installation

```bash
pip install swap-io-py-solana-utils
```

Or install from source:

```bash
pip install git+https://github.com/swap-dot-io/swap-io-py-solana-utils.git
```

## Requirements

- Python >= 3.10
- solders >= 0.18.0
- base58 >= 2.1.1
- pydantic>=2.0
